Book Completion

Edit on GitHub

Overview

The library keeps a record of items you've obtained and monsters you've defeated.
Each entry has its own visual preview and description, some of which can be pretty humorous.

There are a total of 716 items and 366 monsters.
As you can imagine, this is going to take serious effort to reach 100% completion.

  • Item Book
  • Monster Book

Item Book

General

The shortest path is to use Interim Adventures and Rescue Passwords for rare items.

Clear Reward or Side Quest

Weapon
  • Bargain Blade - Bargain Hunter NPC
  • Sturdy Hammer - Banana Kingdom (1st clear, Elite Shop)
  • Wonder Pick - Platinum Paradise (Treasure Shrine)
  • True Knife - Level Borg Mamel to Lv99.
  • Fuuma Sword - Upgrade a Storied Katana+99 at the Monkey Village blacksmith.
  • Kaburagi - There are 2 ways to obtain it.
    • Upgrade a S-Class Kabura+99 using the Smith 2 NPC.
    • Deposit 99,999,999 Gitan into your bank account.
  • Kamina's Sword - Trial Road (Treasure Shrine)
Shield
  • Parry Shield - Static Forest (Clear)
  • Nirvana Board - Jaguar's Hunting Forest (Clear)
  • Helix Shield - There are 2 ways to obtain it.
    • Upgrade a Legendary Fuuma+99 using the Smith 1 NPC.
    • Double Strike Cave (Clear)
  • Jaguar's Shield - Bayside Monster Cave (Clear)
Other
  • Bulldog Bracelet - Cave of Endless Night (Clear)
  • Hemoji Arrow - Platinum Paradise (Boss Fight)
  • Piece of Paper - Let a Gyairas peck a scroll or talisman.
  • Wet Scroll - Use a Water Pot on a scroll.
  • Time Switch Trap - Jaguar's Hunting Forest

Pick-A-Choice

Some dungeons that allow carry-in items can generate Pick-A-Choice shops with rare prizes.
See Always Win Pick-A-Choice to learn how to cheat at Pick-A-Choice.

  • Cave of Endless Night - Upgrade Pot, Degrade Pot, Angel Seed, Scout Bracelet, Fortress Staff
  • Jaguar's Hunting Forest - Calling Pot, Fortress Staff, Eradicate Scroll, Extinction Scroll, Night-Day Scroll

Monster Book

General

  • Night monsters limit break when they level up, so these staves cannot be used.
